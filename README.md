#  Secure NGO System

A Flask-based web application for secure message encryption and management using RSA algorithm.

##  Features
- User Registration & Login
- RSA-based Message Encryption
- Secure Message Storage (SQLite)
- Message Decryption System

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS (Bootstrap)

##  How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open in browser:
   http://127.0.0.1:5000/
